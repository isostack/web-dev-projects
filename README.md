# Front-End Development Portfolio

This repository showcases a collection of standalone front-end projects built with HTML, CSS, and JavaScript.

## Technical Skills Demonstrated

- HTML5 semantic structure and multi-page site architecture
- CSS3 layout and styling (positioning, typography, responsive-friendly structure)
- JavaScript DOM interaction and form behavior
- Media embedding and asset integration (images, video, and supporting files)
- Cross-file project organization and maintainable folder structures
- Front-end debugging and iterative refactoring

## Selected Project Work

- `sblogger-news`  
  Blog/news interface with supporting styles and form scripting.  
  Key skills: page layout, reusable CSS, JavaScript form handling.

- `daily-sudoku`  
  Grid-based Sudoku interface and styling work.  
  Key skills: structured markup, CSS layout precision.

- `kansas-elections-map`  
  Multi-page interactive map site with district pages.  
  Key skills: navigation across linked pages, visual content structure.

- `ici-cryptography`  
  Educational multi-page website on cryptography topics.  
  Key skills: consistent site theming, information architecture.

- `skyweb-planisphere`  
  Planisphere/sky map project with supporting scripts and styles.  
  Key skills: JS + CSS integration, media-rich page composition.
